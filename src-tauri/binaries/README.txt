You have to download the arduino-cli binaries at https://arduino.github.io/arduino-cli/0.35/installation/
Please put the one you need in this folder, with this format :

arduino-cli-<SYSTEM TRIPLET> 
for instance, on mac intel (which triplet's is x86_64-apple-darwin) : arduino-cli-x86_64-apple-darwin

find more on triplet on 
https://doc.rust-lang.org/nightly/rustc/platform-support.html
